# More Commands & Tools

## Quick & Simple HTTP Server

This will serve the files in the current directory from a web server.

```
BabyMac:~ jennapederson$ python -m SimpleHTTPServer
```

Now navigate to [http://localhost:8000](http://localhost:8000).
